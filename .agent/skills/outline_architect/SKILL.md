# 长篇连载大纲规划师 (Serial Novel Outline Architect)

## 0. 核心铁律：节奏控制 (ABSOLUTE RULE: Pacing Control)

> **历史教训**：曾在第二卷大纲中试图一卷内解决全部矛盾（建势力+打内战+考古+逆向投递+最终决战），导致节奏崩溃、规模失真。长篇连载的大纲必须像"挤牙膏"而非"泼水"。

### 0.1 单卷信息量上限
- 每一卷**只允许推进1-2个核心矛盾**，绝不能一卷清盘
- 每一卷必须**留下至少2个未解悬念**作为下一卷的钩子
- 一个大Boss的阴谋从露头到最终解决，**至少跨越3卷以上**
- 角色的成长弧线必须**缓慢且可信**，不允许一卷内从底层直升顶层

### 0.2 "三幕五阶段"卷内结构
每一卷40章，按以下节奏分配：

| 阶段 | 章数 | 占比 | 功能 |
|---|---|---|---|
| **铺垫与新常态** | 第1-8章 | 20% | 建立本卷的初始状态、新环境、新矛盾的萌芽 |
| **升级与日常** | 第9-18章 | 25% | 日常互动+能力小幅提升+次要矛盾的展开 |
| **转折与危机** | 第19-28章 | 25% | 核心矛盾激化、关键信息揭露、信任/关系考验 |
| **高潮** | 第29-35章 | 17.5% | 主线冲突的集中爆发、生死关头 |
| **余波与钩子** | 第36-40章 | 12.5% | 本卷收束+埋下下一卷的重磅伏笔/悬念 |

### 0.3 反面教材（绝对禁止的大纲错误）

❌ **规模膨胀**：主角身边突然出现几千人的势力（必须与前文的世界体量一致）
❌ **一卷全清**：一卷内完成"发现阴谋→集结力量→正面决战→击败Boss"的全流程
❌ **人设跳变**：底层杂役在一卷内变成领袖/强者（至少需要2-3卷的渐进积累）
❌ **支线吞噬主线**：内门政治斗争等支线抢占了跨时空互动的篇幅
❌ **凭空加人**：突然出现大量从未出现过的名字作为关键角色

## 1. 全书架构规划 (Full Series Architecture)

### 1.1 卷间递进原则

在规划多卷大纲时，必须确保每一卷相比前一卷在以下维度有**可感知但适度**的提升：

| 维度 | 递进方式 | 注意事项 |
|---|---|---|
| **世界范围** | 从宗门内部 → 北荒区域 → 跨大陆 | 每卷只扩展一个层级，不能跳级 |
| **敌人层级** | 从宗门内鬼 → 区域势力 → 远古存在 | Boss不能一卷内从出场到被击败 |
| **角色关系** | 从利益交换 → 信任 → 羁绊 → 生死相依 | 感情线必须自然递进，不能突变 |
| **角色实力** | 每卷最多提升1-2个小境界 | 修仙等级膨胀是大忌 |
| **阴谋揭露** | 每卷只揭开冰山一角 | 留白比解释更有力 |

### 1.2 伏笔链管理

大纲必须维护一张**伏笔追踪表**：

```
| 伏笔ID | 埋设位置 | 内容 | 预计触发卷 | 状态 |
|--------|---------|------|-----------|------|
| F001   | 第1卷·第7章 | 紫气灵泉的真实来源 | 第1卷·第29章 | ✅ 已触发 |
| F002   | 第1卷·第40章 | 地脉后门密钥 | 第2卷 | 待触发 |
| ...    | ...     | ...  | ...       | ...  |
```

**规则**：
- 每卷必须**触发**至少2个前置伏笔
- 每卷必须**埋设**至少3个新伏笔（其中至少1个跨2卷以上才触发）
- 伏笔不能无限堆积：超过3卷未触发的伏笔必须在大纲中标注计划触发时间

### 1.3 双线平衡法则

本小说采用"古代线（苏清月）+ 现代线（林凡）"双线叙事。大纲中必须确保：

- 每一卷中，两条线的**篇幅比例不超过6:4**（无论哪条是主线）
- 两条线必须有**至少3个交汇点**（通过传音阵互动的关键节点）
- 如果一条线暂时断联，单线运行**不得超过5章**，否则读者会遗忘另一条线
- 每条线必须有**独立的矛盾和推进**，不能让一条线沦为另一条线的被动等待

## 2. 世界体量自洽检查 (World Scale Consistency)

### 2.1 数字敏感度

在大纲中涉及任何数量描写时，必须与已建立的世界观交叉验证：

- **宗门规模**：没落时期的青云宗，全宗上下（含杂役）**不超过百人**级别。掌门半步结丹，连绩效奖金都发不出，不可能养几千人
- **灵石经济**：一块下品灵石就能引发冲突的贫穷社会。涉及任何资源数量时，必须以此为基准
- **修为天花板**：没落时期现代最高修为是半步结丹。任何凭空出现的元婴期强者都需要合理解释来源
- **地理范围**：宗门位于北荒边缘，周围是废土禁区。势力范围的扩展必须有合理的交通/通信手段支撑

### 2.2 角色能力成长曲线

为主角制定**全书成长路线图**，确保不出现突兀的跳跃：

```
第1卷：凝气期 → 凝气后期（靠古代功法补基础）
第2卷：凝气后期 → 筑基初期（靠盲盒资源+实战突破）
第3卷：筑基初期 → 筑基中后期
第4卷：筑基后期 → 结丹初期
第5卷：结丹 → ？（最终决战不一定靠硬实力碾压）
```

> **核心原则**：林凡的优势永远是**信息差和智慧**，而非修为碾压。他在每一卷中的实力提升都应该伴随着代价和合理的契机。

## 3. 大纲书写流程 (Outline Writing Workflow)

### Step 1: 全书骨架（先写五卷一句话概括）
在动手写任何细纲之前，先用**一句话**概括每一卷的核心事件：

```
第1卷：[已完成] 跨时空初遇、日常交流、发现万年阴谋、第一次跨时空协作
第2卷：[一句话] ______
第3卷：[一句话] ______
第4卷：[一句话] ______
第5卷：[一句话] ______
```

确保五句话连起来构成一个**完整的、层层递进的故事弧线**。

### Step 2: 卷间衔接检查
写完骨架后，逐一检查相邻两卷之间：
- 上一卷的**悬念钩子**是否在下一卷的**铺垫阶段**被接住？
- 角色的**情感状态**是否连贯？（不能上一卷生死离别、下一卷开篇就乐呵呵）
- **世界范围**是否自然扩展而非突变？

### Step 3: 单卷细纲
按照第0.2节的"三幕五阶段"框架，为每一卷写出五个阶段的细纲。每个阶段包含：
- **古代线要事**（1-2句话）
- **现代线要事**（1-2句话）
- **跨时空交汇点**（有则标注）
- **触发的伏笔** / **埋设的伏笔**

### Step 4: 强制交叉验证
细纲写完后，必须执行以下检查：
1. ✅ **节奏检查**：本卷是否只推进了1-2个核心矛盾？是否留了悬念？
2. ✅ **体量检查**：人数、资源、势力范围是否与世界观一致？
3. ✅ **伏笔检查**：触发了几个旧伏笔？埋了几个新伏笔？
4. ✅ **双线检查**：两条线的篇幅比是否合理？交汇点够不够？
5. ✅ **人设检查**：角色的行为是否符合其已建立的性格和能力？有无跳变？
6. ✅ **地理检查**：是否与 `world_setting.md` 的地理设定一致？（参见 cultivation_expert SKILL 第3节）

### Step 5: 写入 plot_outline.md
将审核通过的大纲更新到 `plot_outline.md` 中。格式要求：
- 每卷一个二级标题（`##`）
- 每卷下方先写一行 **核心基调**
- 再写一行 **卷头概览**（2-3句话概括本卷主线，不要写创作备忘）
- 然后是五个阶段的分述（含章节范围、内容摘要、看点）

## 4. 大纲维护规则 (Outline Maintenance)

- **大纲是活文档**：每写完一卷的实际章节，必须回来更新大纲，标注哪些内容已执行、哪些需要调整
- **不要提前写太细**：距离当前卷超过2卷的大纲，保持一句话骨架即可，不要过早展开细纲（因为前面的蝴蝶效应会改变后面的走向）
- **读者视角审核**：每次更新大纲后，站在读者角度问自己——"如果我每天追更一章，到了这一卷的结尾，我是否会迫不及待地想看下一卷？"如果答案不确定，说明悬念钩子不够强
