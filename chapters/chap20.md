# 第一卷：时空的回音

## 第二十章：内门招揽与图册上缴

天还没亮透，外门阵法修理司门口就已经挤满了人。

林凡一推门，先闻到的是包子味、热粥味和劣朱砂混在一起的怪味，接着才看见门口那两张破桌。

桌上堆着吃的。

热包子、灵米粥、卤灵蛋、半包果脯、两捆新麻绳，甚至还有人不知从哪儿弄来一小叠边角符纸，整整齐齐码在最上头，生怕他看不见。

门口那帮平时见了他只会催活的外门弟子，这会儿一个个笑得比谁都真诚。

“林哥起了。”

“昨晚睡得可好？”

“我这粥还热着，特意给你留的。”

“林哥，你那套省料法子真神，昨晚主峰都传开了。”

林凡站在门口，眼皮都没抬，先把这一圈人从头看到尾。

昨天他还只是三组一个会修破板子的杂役。

今天这些人就已经学会拿笑脸往他脸上贴了。

李胖子抱着两笼热包子站在最前头，笑得嘴都快咧到耳根。

“凡哥，我就知道你一出来准得被堵。”他一边说一边把包子往林凡怀里塞，“昨儿你那一手把刘长老都震住了，半夜里马脸管事挨骂挨得跟条狗一样。今天一早，内门那边都来人了。”

林凡接过包子，咬了一口，这才淡淡开口：

“来人就来人。你们一大清早堵我门口，是来拜师，还是来投坟？”

众人脸上的笑都僵了一下。

有人赶紧接话。

“林哥说笑了，我们这不是敬你有本事么。”

“是啊，谁不知道你现在是外门第一号能人。”

林凡把手里的包子咽下去，眼神却没软半分。

“有本事的时候叫林哥，活压下来时是不是还叫林凡？”

这话一落，门口顿时安静了。

李胖子都下意识缩了缩脖子。

林凡也没继续往下压，只伸手把桌上那桶还冒热气的灵米粥拎了过来。

“东西我收一口，情分我先不认。”他淡淡道，“往后谁真想跟我说话，先学会一件事。”

“什么事？”有人小心问。

“别空着手讲交情，也别拿交情压我。”

说完这句，他端着粥，径直往修理司里走。

后头那群人你看我、我看你，竟一时没人敢再嬉皮笑脸地往前凑。

李胖子赶紧抱着包子追上去，小声道：

“凡哥，你这也太不客气了。”

“客气有用？”林凡喝了口粥，“我今天若是笑着把这些东西都接了，明天他们就敢笑着把活全扔我头上。”

两人刚进三组那间破办公室，院外便落下一架青灰灵梭。

刘长老先下来，脸上带笑，脚下却走得极快。跟在他身后的，是一身黑金纹法袍的陈子墨，神色冷淡，目光落下来时像一把薄刃，先把整间屋子扫了一遍。

李胖子一看见这阵仗，立刻识趣地往旁边缩。

“林凡。”陈子墨没绕弯子，开口便道，“大长老看过你昨日所交玉简。内门阵法核心司给你留了位置。静室、月例、材料配额，一并可以翻一档。你这种人，继续留在外门修破板子，是浪费。”

李胖子站在一边，听得眼珠子都快掉下来了。

静室。

配额。

内门核心司。

这三样随便拿一样出来，都够外门弟子做半个月白日梦。

刘长老也笑着补了一句：

“你若愿意点头，今天就能随我们走。”

林凡没答。

他先低头看了一眼脚边那只裂口木桶，又抬眼看向桌上那卷还没收好的后山旧图，最后才把目光挪回陈子墨脸上。

“听着不错。”

刘长老脸上的笑更深了。

“你肯去，自然是……”

“可惜我不去。”

屋里一下静了。

李胖子嘴巴都张开了，险些忘了合上。

刘长老的笑也顿住。

陈子墨看着林凡，声音反而更冷了。

“理由。”

“简单。”林凡竖起两根手指，“第一，你们要的是我这套法子，不是我这个人。我要真进了内门，先被你们翻来覆去问清楚的是手艺，后头吃不吃得上饭，反倒得看你们脸色。”

“第二，我这套本事不是在静室里练出来的，是在烂板子、破阵、旧料堆里磨出来的。让我进内门供起来，不出半个月，我这把刀就得先钝。”

陈子墨盯着他。

“所以你宁肯在外门守这堆破烂？”

林凡点头。

“至少这堆破烂归我看得见。”

刘长老忍不住皱眉。

“你可知道多少人想进而不得？”

“知道。”林凡看着他，声音不高，却没有半点退意，“可我也知道一件事。”

“什么？”

“你们现在来要人，不是因为看得起我。”林凡把桌上一块裂阵板拿起来，往桌面一敲，声音发闷，“是因为你们那边已经开始缺会干活的人了。你们怕我留在外门浪费是假，怕我不肯给你们卖命是真。”

李胖子听得头皮发麻。

刘长老的脸色也终于变了。

这种话，哪怕是真的，平日里也没人敢这么摊开了说。

陈子墨却没发怒，反而盯着林凡看了几息，忽然道：

“你胆子倒大。”

“胆子不大，昨天就活不下来。”林凡淡淡回道。

陈子墨沉默片刻，忽然换了个问法：

“那你想要什么？”

刘长老猛地转头看他。

显然，他没想到陈子墨会顺着林凡这句话往下接。

林凡也不客气，直接开口：

“第一，图册我可以交，方法我可以交，但我人留外门。”

“第二，给我外门三组的旧料调用权，后山和外门旧阵的现场查看权。”

“第三，往后主峰若真要借我去看活，不许拿一句‘顺手帮忙’就想把人打发了。人我可以去，材料、记录、工簿，一样不能少。”

这番话说完，连李胖子都听愣了。

这不是拒绝。

这是当着内门的面，直接开价。

刘长老率先冷下脸来。

“你一个外门弟子，胃口不小。”

林凡连眼皮都没眨。

“胃口小的人，昨天死在台上。”

这句话太硬。

硬得连刘长老一时间都没接上。

陈子墨看着林凡，眼里那点轻慢终于淡了些。

他明白了。

林凡不是不识抬举。

他是比谁都清楚，什么时候该低头，什么时候绝不能把脖子伸出去任人套绳。

“图册。”陈子墨伸出手，“先交。”

林凡早有准备，转手从抽屉里拿出一枚劣质玉简，放在了桌上。

“外层变线、省料思路、几处替代手法，都在里头。”他看着陈子墨，“拿去用没问题，想靠这东西把我整个人拆开来抄，不够。”

陈子墨拿起玉简，目光却没离开林凡。

“你觉得没人能学？”

“能学。”林凡答得很平，“可学会画一笔，和知道该在什么地方画这一笔，不是一回事。”

陈子墨的手指在玉简边缘停了一瞬。

他当然听得懂这句话。

林凡最值钱的，不只是那几笔省料变化。

是他看烂局的眼睛。

刘长老显然还不甘心，沉声道：

“你以为自己留在外门，就能一直稳？”

“稳不稳另说。”林凡抬眼看向他，“至少我现在站在这儿，是我自己挑的地方。”

屋里又静了片刻。

最后，陈子墨把玉简收进袖中，转身往外走。

走到门口时，他才像随口提起一般丢下一句：

“主峰千云阵最近有几处旧病反复。你既然舍不得离现场太远，过些天，或许真有你的用处。”

林凡目光微沉。

千云阵。

护山大阵。

主峰这时候来抢人，果然不只是看中他昨天那一手。

是有些地方，已经开始漏风了。

“那就等你们带着真东西来找我。”林凡声音不大，却让门口的陈子墨顿了一下，“别拿名头来，拿阵材、拿图、拿工簿。要不然我去了，也只能看你们表演。”

陈子墨没回头，只丢下两个字：

“狂妄。”

林凡坐回那张破椅子上，淡淡回了一句：

“总比白跑强。”

等灵梭飞远，李胖子才猛地扑到桌边，声音都快劈了。

“凡哥！那可是内门核心司！”

“我知道。”

“你真一点都不心动？”

林凡抬头看了他一眼。

“心动。”

李胖子一愣。

“那你还……”

“因为心动，所以更不能去。”林凡把桌上那卷后山旧图重新收起，“人一旦真被好东西晃了眼，最容易忘了自己现在到底靠什么吃饭。”

李胖子听得半懂不懂，只觉得这话又像有理，又像疯话。

林凡却已经低头，把那半桶热粥喝得一干二净。

“接下来干什么？”李胖子问。

林凡望向后山方向。

晨雾后头，那一线常年盘着的灰黑阴气还压在那里，像山里没睁开的眼。

“先把外门站稳。”他慢慢道，“等他们都以为我只是舍不得这堆烂板烂料的时候，再把手伸进主峰。”

李胖子喉头滚了滚。

“那到时候……”

“到时候谁再想拿我当好使唤的外门杂役，就得先问问自己够不够格。”
