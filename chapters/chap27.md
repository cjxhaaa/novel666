# 第一卷：时空的回音

## 第二十七章：李胖子觉得凡哥忽然值钱了

李胖子最先发现，林凡现在在外门是真的值钱了。

不是因为月例多了多少。

也不是因为有人给他换了大屋子。

真相反，林凡照旧住那间漏风破屋，床板硬，饭碗缺口，外袍也是旧的。

可他走在外门路上，别人看他的眼神，已经一点点变了。

从前是“这人能干活”。

现在是“这人不能得罪”。

李胖子最先察觉到这一点，是在清晨去阵法司领活的路上。

井边那个平日里最会摆脸色的库房老周，竟主动探头出来，陪着笑问：

“林组长，昨儿分出来那批防潮铜钉，若有富余，记得先知会我一声。”

李胖子当时正弯腰提桶，听得差点把绳子扔井里。

老周这种人，平时见了他们连鼻孔都懒得抬。

如今竟然先笑，再叫“林组长”。

这还没完。

再往前，杂役处的小吏也赔着笑递来文簿。

“林组长，东侧偏门那块裂盘，还得劳烦你们组三给看看。”

再往前，连扫阶的杂役都凑上来，小声问三组库里那捆旧麻绳能不能匀三尺出去。

李胖子一路看得眼花缭乱，跟在林凡后头，终于忍不住低声道：

“凡哥，你有没有发现，他们现在对你说话都客气得瘆人？”

林凡翻着工簿，神色淡淡。

“客气归客气，活少了吗？”

“那也不一样啊。”李胖子压着嗓子，“以前他们看咱们，跟看板车轱辘似的，能用就用，用完就踢。现在不一样，现在他们说话前，先看你脸色。”

林凡这才抬眼看了他一下。

“所以你很高兴？”

“那肯定啊。”李胖子嘿了一声，“以前哪有这待遇。”

林凡没再说话，只把工簿一卷，塞回怀里。

等到了午时，李胖子才发现，自己高兴得还是早了。

两人蹲在阵法司外头的石阶上吃午饭。

今天的菜汤比昨日多飘了两片叶子，算是难得的体面。李胖子正低头吸溜，一名穿灰褂的外门弟子便提着个小纸包快步走来。

那人脸上堆着笑，一开口便先叫：

“林组长。”

李胖子耳朵立刻竖了起来。

那人把纸包递过去，神神秘秘地压低声音：

“这是我表兄在主峰符房做杂工时攒下来的边角料，成色一般，可也比外门常见的粗符纸强些。林组长若不嫌弃，就当我请你喝茶。”

林凡接过来看了一眼。

纸包里果然是半沓新符纸边角，裁得不齐，却明显比外门能碰到的料好得多。

李胖子看得眼都直了。

这东西放他们手里，已经算真金白银。

林凡却只把纸重新包好，搁到一边。

“直说。”

那人一愣。

“什么？”

“你不会平白来送东西。”林凡看着他，“想办什么事，直接说。”

那人被他一句话戳穿，也不敢再绕，忙赔笑道：

“就是我那边有块照明小盘总出毛病，回头若轮到林组长手里，能不能先给看看？”

李胖子在一边听懂了。

这不是请客。

这是提前递人情，想走后门。

林凡把纸包往旁边一推。

“东西我先收。盘子若真轮到我这边，我自然会看。可该怎么排，还怎么排。”

那人脸上笑意有些僵，却也不敢再多说，只能点头应是，灰溜溜退了。

李胖子立刻凑上来。

“凡哥，这可是好东西。”

“我知道。”

“那你怎么还……”

“因为东西越好，后头跟着的事越麻烦。”林凡喝了口菜汤，声音很平，“他今天送半包符纸，明天就敢拿这点面子来要我插手排活。我要真松口一次，以后谁都敢带着破纸烂果来堵门。”

李胖子听得一愣一愣的。

他以前只觉得，别人肯送东西就是给脸。

现在才发现，有些脸接了，后头是要还账的。

傍晚时分，阵法司那边果然又来了人。

说外门东墙下一排照明阵盘坏了，夜里那段路太黑，正好三组离得近，问能不能顺手过去修了。

来人说话极客气。

“林组长，您如今是司里有名的稳当人，这等活别人去，大家都不放心。”

李胖子听得胸口一热，差点替林凡应下。

林凡却抬手把工具箱一合，直接问了一句：

“灯油呢？”

来人一愣。

“什么？”

“补照明阵盘，你总不能让我空手吹亮。”林凡抬眼看他，“梯子呢？灯油呢？铜钉呢？”

那人被问得卡了一下。

“司里以往……你们修理组都是顺手……”

“顺手出工，顺手垫料，顺手熬到天黑，最后再顺手听一句‘辛苦’？”林凡看着他，声音一点点冷下来，“回去告诉派你来的人，我不是不去。”

“东西先送来。”

那人脸上的笑明显挂不住了。

“林组长，不过是修几块旧照明盘……”

林凡抬眼，直接打断。

“那你自己去。”

偏院里一下安静下来。

旁边几个本来在忙活的外门弟子都偷偷抬头看了过来。

这种事若放在从前，外头来一句“顺手帮忙”，他们多半就真去做了。

料自己垫。

梯子自己扛。

忙到天黑，也不过换一句“辛苦”。

可现在林凡偏偏不接这套。

来人站在原地，脸上一阵红一阵白，最后还是咬牙回去搬东西了。

半个时辰后，两桶灯油、一架旧梯子、一包短铜钉，真被送到了三组院里。

李胖子看着那一样样东西往地上摆，整个人都热得发麻。

“凡哥。”他压着声音，眼睛都亮了，“你真把东西全要来了。”

林凡拎起工具箱，神情没什么波动。

“我不是要东西。”

“那是什么？”

“是让他们记住，我这张脸现在不是白用的。”

东墙那边风大。

天色擦黑时，墙根已经起了寒意，那排旧照明盘嵌在墙面里，最坏的三块连灯纹都断了，看着像三只瞎掉的眼。

李胖子在下面搬梯子、递铜钉，忙得鼻尖冒汗。

林凡站在半梯上，拆盘、看纹、换活件，动作一气呵成。

“这块盘边裂了，别硬掰。”他头也不回地吩咐，“活纹能续，死纹再换。能花一份料修好的东西，就别给我拆成两份。”

“灯油挪远点，别放我脚下。”

“阿顺，接这块边框。”

墙下好几个外门弟子都在看。

他们白天听着别人一口一个“林组长”“稳当人”，心里还总觉得是客套。如今真看见林凡站在梯上，三下五除二就把那块谁都懒得碰的旧盘拆开、接回、点亮，才终于明白——

这不是白叫的。

第一块盘亮起时，光还有些晃。

第二块便稳了。

第三块最难，主纹断得深，林凡索性把两块死盘里的活件拆出来，硬拼成一块能用的。

等最后一块也亮起来时，那截原本最黑的东墙终于被暖黄灯光照出了一线模样。

李胖子坐在墙根下，望着那一排重新亮起来的灯，心里忽然明白了。

所谓“值钱”，不只是别人嘴上把你捧起来。

而是别人真得捧着料、捧着东西，老老实实来请你出手。

活干完后，林凡跳下梯子，把白天那半包符纸边角随手丢进李胖子怀里。

李胖子一下愣住。

“给我？”

“不然呢？”

“可这是别人送你的……”

“送到我手里，就是料。”林凡收拾着工具，头也不抬，“你不是一直嚷嚷着想学最简单的引火符？拿去练。先把线画稳，什么时候不抖了，再跟我说别的。”

李胖子抱着那包符纸，喉头发紧，半天才用力点头。

“我一定练好。”

林凡抬眼扫了他一下。

“少喊空话，先把梯子扛回去。”

夜里，东墙那排新修好的照明盘把最黑的一截路照出暖色。李胖子回屋后，连饭都顾不上多吃，直接把符纸摊在小桌上，按林凡说的，先练第一笔直线。

第一笔，歪。

第二笔，更歪。

第三笔刚落下，林凡从门外经过，扫了一眼，冷不丁开口：

“你这不是画符，是拿手抖给我示弱？”

李胖子脸一红，立刻把背挺得笔直。

林凡这回没再损他，只抬手把桌上那盏小灯往前推了半寸。

“手腕压住。”他说，“先把第一条线站起来。别的以后再谈。”

李胖子捏紧笔，重重点头。

小屋里重新安静下来，只剩笔尖划过符纸的细响。

而外头那排刚修好的灯，就那么静静亮着，像把整座外门最黑的一段路，先替他们照开了一点。
