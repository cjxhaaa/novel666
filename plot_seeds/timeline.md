# 万年因果对照表 (Cross-Timeline Mapping)

本表用于追踪万年前（古代线）与万年后（现代线）之间已经建立的直接映射。  
当前只记录正史 `chap01` 至 `chap23` 已经坐实的映射关系。

## 时空锚点

- **锚点 A：落星亭**
  - 男女主唯一稳定沟通节点
  - 连接同一物理地点的一万年前后
  - 常态以传声为主，显影为极特殊情况

- **锚点 B：紫气灵泉 / 现代黑雾地洞**
  - 古代线的“祥瑞表象下的异常点”
  - 现代线的“持续外泄黑雾的废土地洞”
  - 是最重要的危机源头锚点

- **锚点 C：藏经与阵图原版 / 残卷**
  - 古代线能提供原版
  - 现代线只能拿到残卷和残史
  - 这是林凡进行“古代给原版，现代做适配”的知识锚点
