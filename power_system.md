# 修炼体系 (Power System)

修士通过吸收天地灵气、运转功法、锤炼肉身与神魂来改造自身。  
本书的力量体系重要，但不会脱离“资源稀缺”和“时代差”这两个核心前提。

## 1. 基础境界

1. **聚气境**：1-9层。能感应灵气并开始在丹田内积蓄。
2. **筑基境**：初期、中期、后期、圆满。灵气更稳定，肉身与经脉开始真正定型。
3. **金丹境**：1-9品。金丹成，才算真正跻身中高阶修士。
4. **元婴境**：破丹成婴，神魂可离体，已属传说级强者。
5. **化神境**：掌握法则雏形，可视作下界理论上的顶级天花板。

## 2. 时代差异

### 万年前（古代线）

- 灵气浓郁，修炼环境完整。
- 化神虽少，但不是只存在于传说里。
- 各种原版功法、阵法、器物和传承链条大体完整。

### 万年后（现代线）

- 灵气极度枯竭，且驳杂、污染、断层并存。
- 现代青云宗明面上的战力天花板以半步结丹为主。
- 元婴已不可考，化神更是神话。
- 修士们为了极小的突破机会，也会对残破灵脉、低阶灵草和旧传承争得头破血流。

## 3. 林凡当前适用的力量逻辑

- 林凡前期的优势不在高境界，而在更会“用穷办法把低境界榨出价值”。
- 他能从苏清月那里获得古代原版知识，再在现代贫灵环境里做适配。
- 这种适配不等于立即暴涨修为，而是更省、更稳、更能活。

## 4. 功法规则

- 功法不仅决定修炼速度，也决定灵气利用方式、经脉承受方式与战斗风格。
- 同一门功法，在古代和现代未必有相同效果。
- 像 `《枯木汲气篇》` 这样的关键法门，其价值依赖于时代差：
  - 在古代，它过于冷门，不够体面，也不适合灵气丰沛环境下的主流修士。
  - 在现代，它反而适合贫灵、驳杂、需要榨干灵机的环境。

## 5. 战力构成

- **灵根**：决定吸纳灵气的效率与偏性。
- **功法**：决定积累速度、经脉负荷与长期上限。
- **阵法/器物理解**：本书里是非常重要的战力来源，尤其对林凡来说更是核心。
- **法宝**：从凡器到高阶法器，差距极大，但前期更强调“合不合手”和“用不用得起”。
- **经验与应变**：在资源匮乏的时代，很多时候比纸面境界更重要。

## 6. 危机相关硬规则

- 黑雾、污染、异变类问题不等于单纯境界压制题。
- 前中期不能通过“主角变强了”就轻易根治这些问题。
- 许多局部胜利，本质上只是阶段压制、隔离或转移风险。
