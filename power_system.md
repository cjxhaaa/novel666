# 修炼体系 (Power System)

修士通过吸收天地灵气，洗髓伐骨，最终逆天改命。

## 1. 境界划分
1.  **聚气境 (Qi Condensation)**：1-9层。初步感应灵气，储存于丹田。
2.  **筑基境 (Foundation Building)**：初期、中期、后期、圆满。灵气化液，铸就丹基。
3.  **金丹境 (Golden Core)**：1-9品。金丹一转，寿命增至五百载。
4.  **元婴境 (Nascent Soul)**：破丹成婴，神魂可离体。
5.  **化神境 (God Transformation)**：掌握天地法则，移山填海。
5.  **化神境 (Deity Transformation)**：掌握天地法则，移山填海。此境界即为当前下界（十地）的极限天花板，修至化神圆满方可尝试飞升九天。

> **时代差异设定**：
> - **万年前（古代）**：灵气浓郁，化神大能虽少但存在，且有飞升通道的零星传说。
> - **万年后（现代）**：灵气极度枯竭。现代修仙界的**绝顶战力仅为“半步结丹”或普通的“金丹初期”**。元婴已是不可考的传说，化神更是神话。因此，现代世界完全断绝了飞升的可能。修士们为了争夺能突破到筑基或金丹的微薄资源（如废弃灵脉、低阶灵草），内卷且不择手段。

## 2. 战力要素
- **灵根**：吸收灵气的效率（金木水火土、雷、冰、风等）。
- **功法**：修炼速度与灵气厚度的决定因素。
- **武技/剑意**：战斗技巧与意志的体现。
- **法宝**：凡器、宝器、灵器、道器、圣器、帝兵。
