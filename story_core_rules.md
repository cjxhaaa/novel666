# 核心硬规则 (Story Core Rules)

> 本文件是本项目最高优先级写作规则。  
> 若与旧大纲、旧人物档案、旧伏笔表或旧写法冲突，一律以本文件为准。

## 0. 框架读取顺序

续写、返工或改纲时，默认按下面顺序看：

1. `story_core_rules.md`
2. `current_story_summary.md`
3. `plot_outline.md`
4. `task.md`
5. `characters.md`
6. `plot_seeds/active_foreshadowing.md`
7. `plot_seeds/timeline.md`

各文件职责如下：

- `story_core_rules.md`：最高优先级硬规则，只管不能碰的边界。
- `current_story_summary.md`：当前正史走到哪了，只认已写正文。
- `plot_outline.md`：后续大方向和卷内阶段节奏。
- `task.md`：第一卷章节落地清单。
- `characters.md`：人物当前状态和关系。
- `plot_seeds/active_foreshadowing.md`：还没收的伏笔。
- `plot_seeds/timeline.md`：古今因果映射。

## 1. 本书真正的核心

- 这是一部跨时空双线修仙成长文，不是侦探破案文。
- 前四卷主驱动是过日子、攒资源、练本事、长关系。
- 第五卷才集中处理探索、拼图、清算和真相闭环。
- 男主在现代前期没有积累，只能靠女主提供的古代原版知识，一点点站稳。
- 女主不是万能外挂，她也会被身份、局势、伤势、人手和时空阻隔限制。

## 2. 传音古阵硬规则

### 2.1 基本定义

- 传音古阵就是“落星亭”。
- 它连接的是同一物理地点的一万年前后，不是随机时空通话。
- 它的本质是跨时空共鸣阵，不是普通通讯法器。

### 2.2 常态能力

- 常态以传声为主。
- 可以传递少量与灵力、阵法、节点相关的共鸣信息。
- 这也是古今对照、原版阵图核对、时空盲盒落点能够成立的根基。

### 2.3 非常态能力

- 稳定传影极难。
- 只有在掌门级权限、原版阵图、强共鸣节点等极特殊条件同时满足时，才可能短暂显出模糊轮廓、节点光影或片段投影。
- 这种显影只能偶发，不能当作常规视频通话。

### 2.4 明确禁止

- 禁止把落星亭写成可移动通讯设备。
- 禁止出现便携式传音玉、耳麦、随身联络阵等替代方案。
- 禁止离开落星亭后继续稳定跨时空通话。
- 禁止无解释地稳定传影、稳定传物、稳定传人。
- 禁止为了推进剧情，临时放宽它的限制。

### 2.5 传音阵的剧情职责

- 它是本书最大的独特卖点之一，不只是工具。
- 它负责支撑男女主最重要的关系黏性。
- 它负责支撑古今信息差、同地时空对照、盲盒埋设与回响。
- 它越珍贵、越受限，男女主的关系就越成立。

## 3. 男女主成长硬规则

### 3.1 林凡

- 前期只能算刚刚站住脚，不能写出“已经有家底”的读感。
- 名头、评定、职位可以先有一点，但随之而来的应该是更重的活、更脏的差事和更高的风险。
- 他的核心优势永远是信息差、适配能力和现代底层生存智慧，不是境界碾压。

### 3.2 苏清月

- 她的优势是见识、修为底子、古代完整认知和身份高度。
- 她不能承担“万能解题器”功能。
- 她每次给出关键帮助时，都要体现出她的局限、代价或盲区。

## 4. 功法与危机硬规则

- `《枯木汲气篇》` 必须坚持“古代冷门、当年少人练、现代已失传、现代反而适用”的逻辑。
- 黑雾、污染、灵泉、地脉问题在前中期只能阶段压制、引流、隔离、封存，不能提前根治。
- 每次压住危机，都应留下新的代价、扩散、误伤或后续麻烦。

## 5. 节奏硬规则

- 第一卷只需要立住时空概念、联盟关系和男主在现代步步为营的感觉。
- 第一卷中段不能写成连续翻账、连续追线、连续拼图。
- 每章先问“这章让角色怎么更会活、怎么更了解彼此、怎么多攒一点东西”，再问“这章是否顺手露出一点问题”。

## 6. 一致性检查清单

每次续写或改纲前，先确认：

- 是否违反了落星亭“同一物理地点”的规则。
- 是否把传音阵写得太方便。
- 是否把前四卷写成了查案剧。
- 是否把林凡写得过早有积累。
- 是否把苏清月写成了万能人物。
- 是否让 `《枯木汲气篇》` 失去了时代差逻辑。
- 是否让黑雾、污染问题在前中期被轻易清盘。
