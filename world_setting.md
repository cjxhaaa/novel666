# 世界观设定 (World Setting)：九天十地

## 1. 地理概貌
九天十地是一个宏大且充满灵气的修仙世界。

### 九天 (The Nine Heavens)
上界强者的领地，灵气浓郁成液，是所有修士向往的终极之地。
- **紫微天**：帝星汇聚，皇朝鼎盛。
- **玄黄天**：古老厚重，多奇珍异宝。
- **青华天**：万木长春，是炼丹师的圣地。
... (其余略，随剧情展开)

### 十地 (The Ten Earths)
广袤且复杂的下界大陆，凡人与底层修士的聚居地。
- **北荒大陆** (故事起点)：灵气较为匮乏，妖兽横行。
- **东海岛群**：散修聚集，遗迹众多。
- **中州圣土**：十地的中心，各大宗门在此争锋。

## 2. 主要势力
- **青云宗 (Azure Cloud Sect)**：自建宗以来一直位于北荒大陆，鼎盛时期势力辐射中州，现已没落。传音古阵"落星亭"连接同一物理地点的万年前后。
- **月神教**：神秘的中州势力，行事诡秘。
- **万妖谷**：妖族的领地，与人界接壤。

## 3. 传说与秘闻
- **星剑传说**：相传万年前，剑祖以此剑劈开鸿蒙，定鼎九天。
- **诸神黄昏**：一个关于众神陨落的古老预言。
